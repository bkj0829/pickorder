동대문 발주 · 입고 관리

기존 엑셀 품목을 불러오고, 주문상품 추가·입고수량 입력·미송/입고완료 자동 분류까지 한 화면에서 봅니다.

입고수량이 발주수량 이상인 항목
전체 발주/입고 리스트
검색, 업체, 상태별로 확인하고 입고수량을 바로 입력합니다.
미송 관리
미입고 수량이 남아있는 항목입니다. 입고수량을 입력하면 자동으로 완료로 이동합니다.
입고완료 목록
발주수량과 입고수량 계산 결과 완료 처리된 항목입니다.
주문상품 추가
엑셀에 없던 주문상품/발주항목을 임시 등록합니다. GitHub 정적 페이지라 브라우저에 저장됩니다.
추가 등록된 주문상품
브라우저에 저장된 추가 주문상품입니다.
업체별 발주/입고 토탈
