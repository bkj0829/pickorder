동대문 발주 · 입고 관리

업체/상품은 한 번만 등록하고, 같은 상품 추가주문은 1차·2차 발주와 발주일시로 구분해 미송과 입고완료를 자동 분리합니다.

로컬 임시저장
업체별 미송 확인
미송은 업체별로 먼저 보고, 업체를 누르면 해당 업체 미송 발주건만 확인합니다.
업체별 토탈
업체별 발주/입고/미송 합계
상품별 토탈
상품별 발주/입고/미송 합계
주문상품 / 옵션 등록
업체와 상품을 선택한 뒤 색상·사이즈·발주수량을 추가하면 발주차수와 발주일시가 붙은 새 발주건으로 저장됩니다.
같은 업체/상품/컬러/사이즈를 다시 등록해도 기존 행을 덮어쓰지 않고 2차·3차 발주건으로 새로 저장됩니다.
입고 처리
입고수량 입력 후 저장하면 확인창으로 한 번 더 체크합니다.
업체별 미송 목록
업체를 선택하면 그 업체의 미송 발주건만 표시합니다.
입고완료 목록
미송수량이 0인 품목입니다.
삭제내역
잘못 주문해서 삭제한 발주건과 삭제사유를 히스토리로 남깁니다.
신상마켓 입고 매칭
신상마켓 입고 상품을 셀메이트 재고현황 상품명과 매칭하고 검품/택 출력을 준비합니다.
1차 버전은 업로드형으로 동작합니다. 신상마켓 입고 JSON과 셀메이트 재고현황 CSV를 업로드하면 화면 안에서 매칭하고 택을 출력합니다.
신상 입고 확인 중
셀메이트 재고 확인 중
매칭 상태 확인 중
검품 리스트 미리보기
입고 수량과 매칭된 셀메이트 상품명을 같이 확인합니다.
택 출력 미리보기
업체명 없이 신상마켓 상품명과 우리 상품명을 같이 표시합니다.
업체 마스터
업체를 미리 등록해서 검색/선택합니다.
상품 마스터
상품명은 한 번 등록하고 옵션만 계속 추가합니다.
구글시트 연동 설정
구글 앱스 스크립트 웹 앱 주소를 넣으면 입력값이 구글시트에 저장됩니다.
아직 웹 앱 주소를 넣지 않으면 이 PC 브라우저에만 임시 저장됩니다. 여러 명이 같이 쓰려면 구글시트 연동이 필요합니다.
설치 순서
처음 연결할 때 이 순서대로 진행하세요.
  1. 구글시트에서 확장 프로그램 → Apps Script를 엽니다.
  2. google_apps_script.gs 내용을 전체 붙여넣고 저장합니다.
  3. 함수 선택에서 초기설정 또는 setup을 선택한 뒤 실행합니다.
  4. 구글 권한 승인 후 시트 하단에 업체마스터 / 상품마스터 / 발주입고내역 / 삭제내역 탭이 생겼는지 확인합니다.
  5. 배포 → 새 배포 → 웹 앱으로 배포합니다. 실행 사용자는 나, 액세스 권한은 링크가 있는 모든 사용자로 설정합니다.
  6. 배포 완료 화면에서 /exec로 끝나는 웹 앱 URL을 복사합니다.
  7. 이 화면 위쪽 웹 앱 URL 붙여넣기 칸에 붙여넣고 연결 저장 → 연결 테스트를 누릅니다.
  8. 연결 성공이 나오면 기존 엑셀 데이터 구글시트 업로드를 한 번만 누릅니다.
