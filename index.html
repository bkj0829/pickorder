동대문 발주 · 입고 관리

업체/상품 마스터를 먼저 잡고, 같은 상품에는 색상·사이즈 옵션만 추가해서 발주·입고·미송을 자동 계산합니다.

원본 엑셀생성일
미송 우선 확인
발주 대비 입고가 덜 된 항목
입고완료 최근 목록
입고수량이 발주수량 이상인 항목
전체 발주/입고 리스트
검색, 업체, 상태별로 확인하고 입고수량을 바로 입력합니다.
주문/옵션 빠른 등록
업체와 상품을 선택한 뒤 컬러·사이즈·수량만 계속 추가합니다.
입력 흐름: 업체 선택/검색 → 상품 선택/검색 → 컬러/사이즈/발주수량만 입력 → 등록. 같은 상품을 계속 넣을 때는 상품명·업체명을 다시 입력하지 않아도 됩니다.
선택 상품 기존 옵션
동일 상품의 기존 컬러/사이즈를 확인하고 누락 옵션만 추가합니다.
추가 등록된 주문상품
브라우저에 저장된 추가 주문상품입니다.
업체 마스터 등록
자주 쓰는 업체를 미리 등록해서 검색 선택합니다.
상품 마스터 등록
상품명은 한 번만 등록하고 이후 옵션만 추가합니다.
등록된 업체 마스터
엑셀에서 불러온 업체 + 추가 등록 업체
등록된 상품 마스터
이 상품으로 바로 발주 등록할 수 있습니다.
미송 관리
미입고 수량이 남아있는 항목입니다. 입고수량을 입력하면 자동으로 완료로 이동합니다.
입고완료 목록
발주수량과 입고수량 계산 결과 완료 처리된 항목입니다.
업체별 발주/입고 토탈
업체별 발주합계, 입고합계, 미송합계, 입고율을 확인합니다.
상품별 발주/입고 토탈
주문상품/사입명 기준으로 합산한 내역입니다.
