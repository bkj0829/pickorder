CSV 백업전체/미송/입고완료 목록을 CSV로 내려받을 수 있습니다.

동대문 발주 · 입고 관리

업체/상품을 미리 등록하고, 같은 상품은 컬러·사이즈·수량만 추가해서 미송과 입고완료를 자동 분리합니다.

로컬 임시저장
미송 우선 확인
발주수량 - 입고수량 기준으로 아직 남은 품목입니다.
업체별 토탈
업체별 발주/입고/미송 합계
상품별 토탈
상품별 발주/입고/미송 합계
주문상품 / 옵션 등록
업체와 상품을 선택한 뒤 색상·사이즈·발주수량만 추가합니다.
입고 처리
입고수량을 수정하면 미송과 입고완료가 자동 계산됩니다.
미송 목록
입고가 덜 된 품목만 표시합니다.
입고완료 목록
미송수량이 0인 품목입니다.
업체 마스터
업체를 미리 등록해서 검색/선택합니다.
상품 마스터
상품명은 한 번 등록하고 옵션만 계속 추가합니다.
Google Sheet 연동 설정
Apps Script 웹 앱 URL을 넣으면 입력값이 Google Sheet에 저장됩니다.
아직 URL을 넣지 않으면 이 PC 브라우저에만 임시 저장됩니다. 여러 명이 같이 쓰려면 Google Sheet 연동이 필요합니다.
설치 순서
README.md에도 같은 내용이 있습니다.
  1. Google Sheet 생성
  2. 확장 프로그램 → Apps Script → google_apps_script.gs 전체 붙여넣기
  3. setup 함수 한 번 실행 후 권한 승인
  4. 배포 → 새 배포 → 웹 앱
  5. 실행 사용자: 나 / 액세스 권한: 링크가 있는 모든 사용자
  6. 웹 앱 URL을 이 화면에 붙여넣고 저장
  7. 처음 한 번만 기존 엑셀 데이터 업로드
